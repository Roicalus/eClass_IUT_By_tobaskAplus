## Privacy Policy & Terms (Firebase Hosting)

This project includes static policy pages used for Google OAuth consent/verification:

- `/privacy` (served from `web/privacy.html`)
- `/terms` (served from `web/terms.html`)

They are included in the Hosting build output (`build/web`).

Deploy:

- `flutter build web`
- `firebase deploy --only hosting`
